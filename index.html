Banking System with Personalized Credit Card Recommendations

The motivation behind doing this project is my background in business due to my minors in Business Administration and Finance. As stated in my About Me, there is nothing I love more than connecting my interests together. On top of that, in regards to this project, my partner is also studying business on top of computer science, so we both put our minds together to come up with this banking system that can provide recommendations on credit cards.

This project has two primary base-level functionalities. The first is the ability to create bank accounts that can contain both savings and checking accounts. In these accounts, users are allowed to withdraw, deposit, and create more accounts. Associated with each account is the user's credit score, age, initial deposit, and name. Then, users are able to prompt the bank for credit card recommendations. This leads to the second functionality of the program. Using the data from these bank accounts, this project takes in data and weighs credit score, age, the amount of money in their accounts, as well as their yearly salary.

This code will be coded strictly in Java for simplicity, and has a variety of classes. On top of static classes, an AbstractAccount class is used to implement the Account class in order to differentiate between Savings and Checking accounts. Also, in order to process transactions, there is a transactional interface. Some of the other classes include: Bank.java, CheckingAccount.java, CreditCardRecommendation.java, and SavingsAccount.java. This structure was decided as it provides the most simple solution to coding our project while still maintaining clean code principles and encouraging simplicity and correctness in development.
